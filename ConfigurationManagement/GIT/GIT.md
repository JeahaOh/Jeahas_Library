# GIT

## git 커밋 이후 변경 파일 되돌리기
- git reset --hard

## git tracking list에서 파일 삭제하기
- git rm --cached `<filename>`

## git tracking list 확인하기
- git ls-files

## .gitignore 만들어 주는 툴
- `https://www.gitignore.io/`
  - 필요한 파일들 까지 ignore 시켜버리니 확인 잘 해야함.

## git branch 생성후 바로 checkout
- git checkout -d `<branchName>`
